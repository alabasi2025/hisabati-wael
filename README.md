# النظام المحاسبي الحديث - Hisabati

## نظرة عامة
نظام محاسبي متكامل تم تحويله بالكامل من Oracle Forms إلى نظام ويب حديث يعمل على Cloudflare Pages + Workers مع قاعدة بيانات D1 (SQLite).

## الميزات المكتملة

### 1. لوحة التحكم (Dashboard)
- إحصائيات شاملة (رصيد الصندوق، القبض، الصرف، القيود)
- مؤشرات KPI ذكية مع تنبيهات
- أرصدة البنوك
- رسم بياني شهري للحركات (Chart.js)
- رسم دائري لتوزيع الحسابات
- أعلى 10 أرصدة
- آخر القيود والسندات

### 2. دليل الحسابات (Chart of Accounts)
- عرض شجري تفاعلي مع Drag & Drop
- عرض جدولي بديل
- بحث لحظي بالرمز أو الاسم
- فلترة بالنوع والمستوى
- إضافة/تعديل/حذف حسابات
- نقل حسابات بين الفروع
- حتى 4 مستويات هرمية

### 3. القيود اليومية (Journal Entries)
- إنشاء/تعديل/حذف/نسخ/عكس القيود
- ترحيل القيود مع تحديث أرصدة الحسابات
- دعم متعدد العملات مع سعر الصرف
- **دعم مراكز التكلفة لكل سطر قيد**
- قوالب جاهزة (رواتب، إيجار، مبيعات، مشتريات، مصاريف)
- فلترة بالحالة والتاريخ

### 4. سندات القبض والصرف (Vouchers)
- إنشاء/تعديل/حذف/نسخ/عكس السندات
- ترحيل تلقائي مع إنشاء قيد محاسبي
- دعم طرق الدفع (نقد، شيك، تحويل)
- بيانات الشيك (رقم، تاريخ، بنك)
- **ربط السند بمركز تكلفة**
- قوالب سندات جاهزة
- تصدير PDF

### 5. التقارير
- **ميزان المراجعة** - فلترة بالتاريخ والمستوى
- **كشف حساب** - مع رصيد جاري
- **قائمة الدخل** - إيرادات ومصروفات وصافي ربح/خسارة
- **الميزانية العمومية** - أصول وخصوم وملكية مع فحص التوازن
- **تقرير مراكز التكلفة** - أرصدة المراكز مع حركات القيود والسندات
- تصدير جميع التقارير (PDF, Excel, CSV)
- طباعة احترافية

### 6. مراكز التكلفة (Cost Centers)
- إنشاء/تعديل/حذف مراكز
- ربط بالقيود المحاسبية
- ربط بالسندات
- تقرير أرصدة شامل
- عرض حركات كل مركز بالتفصيل

### 7. إدارة النظام
- **المستخدمين** - إنشاء/تعديل مع أدوار متعددة (مدير، محاسب، مستخدم، مشاهد)
- **الصلاحيات** - صلاحيات تفصيلية لكل وحدة (عرض، إنشاء، تعديل، حذف، طباعة)
- **العملات** - إدارة العملات وأسعار الصرف
- **السنوات المالية** - إنشاء/تفعيل/إغلاق مع ترحيل أرصدة
- **إعدادات النظام** - 5 تبويبات (الشركة، مالية، ترقيم، طباعة، النظام)
- **سجل النشاطات** - تتبع كامل لجميع العمليات

### 8. إقفال السنة المالية
- إقفال حسابات الإيرادات والمصروفات
- ترحيل صافي الربح/الخسارة للأرباح المحتجزة
- تحديث الأرصدة الافتتاحية
- إنشاء قيد إغلاق تلقائي
- تقرير نتائج الإغلاق

### 9. ميزات عامة
- تصميم RTL عربي احترافي (Dark Mode)
- بحث شامل (Ctrl+K)
- نظام إشعارات
- طباعة احترافية مع ترويسة الشركة
- تصدير PDF/Excel/CSV
- Drag & Drop في شجرة الحسابات
- اختصارات لوحة المفاتيح

## البنية التقنية

### Stack
- **Backend**: Hono v4 (TypeScript) on Cloudflare Workers
- **Database**: Cloudflare D1 (SQLite)
- **Frontend**: Vanilla JS + TailwindCSS CDN
- **Charts**: Chart.js
- **Excel**: SheetJS (XLSX)
- **PDF**: jsPDF + html2canvas
- **Icons**: Font Awesome 6

### هيكل الجداول
| الجدول | الوصف |
|--------|-------|
| settings | إعدادات النظام |
| fiscal_years | السنوات المالية |
| currencies | العملات |
| users | المستخدمين |
| user_permissions | صلاحيات المستخدمين |
| modules | وحدات النظام |
| accounts | دليل الحسابات (شجري) |
| journal_entries | القيود المحاسبية |
| journal_entry_lines | تفاصيل القيود + مراكز التكلفة |
| vouchers | السندات (قبض/صرف) + مركز التكلفة |
| voucher_details | تفاصيل السندات |
| cost_centers | مراكز التكلفة |
| audit_log | سجل النشاطات |

### نقاط الدخول (API Endpoints)
| المسار | الوصف |
|--------|-------|
| POST /api/auth/login | تسجيل الدخول |
| GET /api/auth/me | التحقق من الجلسة |
| GET/POST/PUT/DELETE /api/accounts | دليل الحسابات |
| GET/POST/PUT/DELETE /api/journal | القيود اليومية |
| POST /api/journal/:id/post | ترحيل قيد |
| GET/POST/PUT/DELETE /api/vouchers | السندات |
| POST /api/vouchers/:id/post | ترحيل سند |
| GET /api/reports/trial-balance | ميزان المراجعة |
| GET /api/reports/account-statement/:id | كشف حساب |
| GET /api/reports/income-statement | قائمة الدخل |
| GET /api/reports/balance-sheet | الميزانية العمومية |
| GET/POST/PUT/DELETE /api/cost-centers | مراكز التكلفة |
| GET /api/cost-centers/report | تقرير مراكز التكلفة |
| GET /api/cost-centers/:id/transactions | حركات مركز تكلفة |
| GET /api/dashboard/stats | إحصائيات لوحة التحكم |
| GET /api/search?q= | بحث شامل |
| /api/admin/* | إدارة النظام |
