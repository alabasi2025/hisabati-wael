# النظام المحاسبي الحديث - Hisabati v4.5

## نظرة عامة
نظام محاسبي متكامل تم تحويله بالكامل من Oracle Forms إلى نظام ويب حديث يعمل على Cloudflare Pages + Workers مع قاعدة بيانات D1 (SQLite).

## الميزات المكتملة

### 1. لوحة التحكم (Dashboard)
- إحصائيات شاملة (رصيد الصندوق، القبض، الصرف، القيود)
- مؤشرات KPI ذكية مع تنبيهات
- أرصدة البنوك
- رسم بياني شهري للحركات (Chart.js)
- رسم دائري لتوزيع الحسابات
- أعلى 10 أرصدة
- آخر القيود والسندات

### 2. دليل الحسابات (Chart of Accounts)
- عرض شجري تفاعلي مع Drag & Drop
- عرض جدولي بديل
- بحث لحظي بالرمز أو الاسم
- فلترة بالنوع والمستوى
- إضافة/تعديل/حذف حسابات
- نقل حسابات بين الفروع
- حتى 4 مستويات هرمية

### 3. القيود اليومية (Journal Entries)
- إنشاء/تعديل/حذف/نسخ/عكس القيود
- ترحيل القيود مع تحديث أرصدة الحسابات
- دعم متعدد العملات مع سعر الصرف
- دعم مراكز التكلفة لكل سطر قيد
- قوالب جاهزة (رواتب، إيجار، مبيعات، مشتريات، مصاريف)
- فلترة بالحالة والتاريخ
- تصدير PDF لكل قيد

### 4. سندات القبض والصرف (Vouchers)
- إنشاء/تعديل/حذف/نسخ/عكس السندات
- ترحيل تلقائي مع إنشاء قيد محاسبي
- **دعم متعدد العملات (اختيار عملة + سعر صرف)**
- دعم طرق الدفع (نقد، شيك، تحويل)
- بيانات الشيك (رقم، تاريخ، بنك)
- ربط السند بمركز تكلفة
- قوالب سندات جاهزة (إيداع نقدي، تحصيل شيك، صرف رواتب، دفع فاتورة...)
- **عرض تفصيلي مع أزرار نسخ/عكس/طباعة/PDF**
- تصدير PDF للسندات
- طباعة مباشرة مع ترويسة الشركة

### 5. التقارير
- **ميزان المراجعة** - فلترة بالتاريخ والمستوى مع تصدير (PDF, Excel, CSV, طباعة)
- **كشف حساب** - مع رصيد جاري وتصدير متعدد الصيغ
- **قائمة الدخل** - إيرادات ومصروفات وصافي ربح/خسارة مع تصدير
- **الميزانية العمومية** - أصول وخصوم وملكية مع فحص التوازن وتصدير
- **تقرير مراكز التكلفة** - أرصدة المراكز مع حركات القيود والسندات
- **تقرير إقفال نهاية السنة** - مراجعة شاملة للإيرادات والمصروفات والميزانية مع خطوات الإقفال
- تصدير جميع التقارير (PDF, Excel, CSV)
- **طباعة احترافية مع اسم الشركة الديناميكي من الإعدادات**

### 6. مراكز التكلفة (Cost Centers)
- إنشاء/تعديل/حذف مراكز
- ربط بالقيود المحاسبية
- ربط بالسندات
- تقرير أرصدة شامل مع عدد القيود والسندات
- عرض حركات كل مركز بالتفصيل

### 7. إدارة النظام
- **المستخدمين** - إنشاء/تعديل مع أدوار متعددة (مدير، محاسب، مستخدم، مشاهد)
- **الصلاحيات** - صلاحيات تفصيلية لكل وحدة (عرض، إنشاء، تعديل، حذف، طباعة)
- **العملات** - إدارة العملات وأسعار الصرف (IQD, USD, EUR, TRY)
- **السنوات المالية** - إنشاء/تفعيل/إغلاق مع ترحيل أرصدة
- **إعدادات النظام** - 5 تبويبات (الشركة، مالية، ترقيم، طباعة، النظام)
- **سجل النشاطات** - تتبع كامل لجميع العمليات مع فلترة

### 8. إقفال السنة المالية
- إقفال حسابات الإيرادات والمصروفات
- ترحيل صافي الربح/الخسارة للأرباح المحتجزة
- تحديث الأرصدة الافتتاحية
- إنشاء قيد إغلاق تلقائي
- تقرير نتائج الإغلاق التفصيلي

### 9. ميزات عامة
- تصميم RTL عربي احترافي (Dark Mode)
- بحث شامل (Ctrl+K) عبر الحسابات والقيود والسندات
- نظام إشعارات
- **تصدير PDF احترافي مع اسم الشركة الديناميكي**
- **طباعة مباشرة مع ترويسة ديناميكية**
- تصدير Excel/CSV
- Drag & Drop في شجرة الحسابات
- اختصارات لوحة المفاتيح
- Middleware للصلاحيات على جميع المسارات الحساسة

## البنية التقنية

### Stack
- **Backend**: Hono v4 (TypeScript) on Cloudflare Workers
- **Database**: Cloudflare D1 (SQLite)
- **Frontend**: Vanilla JS + TailwindCSS CDN
- **Charts**: Chart.js 4.4
- **Excel**: SheetJS (XLSX) 0.18.5
- **PDF**: jsPDF 2.5.1 + html2canvas 1.4.1
- **Icons**: Font Awesome 6.4

### Middleware الأمان
- **authMiddleware** - التحقق من التوكن (base64 مع صلاحية 24 ساعة)
- **requireRole** - التحقق من دور المستخدم (admin/manager)
- **checkModulePermission** - فحص صلاحية الوحدة تلقائياً حسب HTTP Method
- **requirePermission** - فحص صلاحية محددة على وحدة
